# hacker_news_catcher

Hacker News Downloader using Instapaper(Top 30 news of the trend)

## Prerequisite
- Ruby 2.3 or later
- Bundler
- Instapaper account

## How To Use

```
bundle install
$EDITOR .env
bundle exec ruby hacker_news_catcher.rb
```

## Environment

Edit `.env` file that looks like:

```
INSTAPAPER_EMAIL="foo@bar.baz"
INSTAPAPER_PASSWORD="foobarbaz"
```
